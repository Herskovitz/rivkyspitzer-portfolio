# Fetch API

This folder contains a simple example demonstrating how to fetch data from an API using JavaScript and TypeScript concepts.

## Highlights

- Use of the Fetch API to retrieve data asynchronously.
- Basic integration of API data into a web page.
- Demonstrates handling of promises with `async`/`await`.

---

Open the file in a modern browser to see the Fetch API in action and review the code for a straightforward implementation of API data fetching.
