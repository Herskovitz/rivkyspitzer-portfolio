# Web Projects

This folder contains web development projects and exercises covering key front-end technologies learned in the course.

## Contents

- **HTML-CSS-CafeWeb**  
  A project focused on foundational HTML and CSS skills to create a styled café website, emphasizing semantic markup and custom styling without frameworks.

- **Bootstrap-CafeWeb**  
  A project demonstrating responsive web design using Bootstrap. It showcases practical use of Bootstrap classes and layout techniques to build a mobile-friendly café website.

- **JavaScript**  
  A collection of JavaScript files containing exercises and homework assignments that reinforce core JavaScript concepts and DOM manipulation learned in the course.

---

These projects highlight progressive front-end development skills, from basic markup and styling to responsive design and interactive scripting.
