# Mobile Applications

This folder contains MAUI projects designed to reinforce fundamental skills in XAML and basic app UI construction.

## Contents

- **MauiBasics**  
  A straightforward app illustrating core XAML elements and layouts, organized into tabs to clearly demonstrate individual UI concepts.

- **CalculatorMaui**  
  A simple calculator app reimplemented in MAUI, providing practical experience in creating functional, user-friendly interfaces.

---

These projects focus on strengthening essential MAUI development skills that form the foundation for more complex applications.
