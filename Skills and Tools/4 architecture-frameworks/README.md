# Architecture Frameworks

This folder contains foundational frameworks developed to support application architecture and user interface development within the CPU Code School projects.

## Contents

- **CPUFramework**  
  A core framework providing architectural patterns, utilities, and base components to facilitate consistent and scalable application development.

- **CPUWindowsFormsFramework**  
  A specialized framework built on top of CPUFramework tailored for creating Windows Forms desktop applications with streamlined UI and event handling support.

---

These frameworks demonstrate practical implementation of software architecture principles and reusable code design to accelerate application development across different projects.
