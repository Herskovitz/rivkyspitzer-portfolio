# Hangman Project

This project contains multiple implementations and related materials for the Hangman game.

## Folder Structure

- **HangmanApp**  
  - **DesktopApp** — Desktop application implementation using object-oriented programming  
  - **MauiApp** — MAUI application implementation following object-oriented principles  
  - **HangmanSystem** — Core system components and logic for the Hangman game  
  - **HangmanWeb** — Web application implementation of the Hangman game  

- **spec** — Specification documents related to the Hangman project  
- **UML Diagram** — UML diagrams illustrating the object-oriented design of the project
