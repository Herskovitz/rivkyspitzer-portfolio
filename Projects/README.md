# Projects

This folder contains two main projects developed during the CPU Code School course. Together, they cover the wide range of concepts, tools, and skills taught throughout the program.

## Projects Overview

- **Recipe App**  
  A desktop application focused on data management and backend functionality, including unit testing and database integration.

- **Hangman Game**  
  A multi-platform game with web, desktop (WinForms and MAUI), and object-oriented programming implementations, showcasing UI design and application logic.

## Skills and Tools Covered

Combined, these projects demonstrate proficiency in:

- Software design and OOP principles
- Desktop and web application development
- Unit testing and exception handling
- Database design and CRUD operations
- Cross-platform development with MAUI
- User interface and interactive web technologies
