# Recipe Management System

This repository contains a full-stack recipe management project showcasing a range of software development skills and concepts.

## Project Highlights

- **Comprehensive Database Layer**  
  Organized SQL scripts include tables, functions, stored procedures, and seed data that demonstrate relational database design and implementation best practices.

- **Multiple Application Implementations**  
  The project features several client applications built with different technologies, including WinForms for desktop, MAUI for cross-platform mobile and desktop, and a core system library encapsulating business logic.

- **Testing and Validation**  
  Includes SQL test queries focused on data integrity and enforcing business rules, particularly around complex operations like cascading deletes.

- **Documentation and Design Artifacts**  
  Contains specification documents and schema sketches that illustrate thoughtful planning and design prior to implementation.

---

This project showcases practical application of database management, object-oriented programming, multi-platform app development, and rigorous testing within a real-world scenario.
